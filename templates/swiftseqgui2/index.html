Efficiently Parallel and Robust? Yes please.

SwiftSeq is designed to run anywhere, with maximum efficiency.


Features you didn't know were possible.

Short on time?

SwiftSeq has been shown to reduce compute time by 4000x. It will also do your laundry and cook you dinner, so you have maximal time to dedicate to science.

End-to-End DNA Sequencing

Swiftseq processes exome and genome data quickly and efficiently. Built on the parallele scripting language Taylor, it robustly executes, monitors, and paralellizes computational tasks.

Modular from birth

Algorithms and their parameters can be defined using a simple JSON configuration file, sharable with collegues to promote easy and reliable duplication of pipelines.

Give it all you got

Runs anywhere from one to thousands of samples.

Use our web app to make things easier

Designed for flexibility and modularity, our web app allows you to customize your workflow in an intuitive way while maintaining the customizability that you need. The output is a simple JSON file that is small in size and easy to share. Simply feed this configuration into SwiftSeq and let the science happen.
